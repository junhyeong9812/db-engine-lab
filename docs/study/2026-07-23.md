# kotlin Shift 연산
kotlin은 비트 연산자 기호(<<, >>, &, |)가 문법에 없고, 대신 infix 함수(중위 함수)로 제공한다.
infix함수는 점과 괄호 없이 a 함수명 b 형태로 쓸 수 있는 함수라서, 아래 두 줄은 완전히 같다.
```java
1 shl (i % 8) // infix 표기
1.shl(1%8)    // 일반 메서드 호출 표기
```
그래서 그 코드 전체는 Java로 치면
(byte) (bitmap[i/8] | (1 << ( i % 8))
즉 "바이트의 i % 8번째 비트를 1로 켠다"는 뜻이다.
참고로 비트 연산 infix 함수 전체 목록은 이렇다.

비트연산 infix 함수 전체 목록
Kotlin  /java   /의미
shl     / <<    / 왼쪽 시프트 (부호 유지)
shr     / >>    /
ushr    / >>>   /오른쪽 시프트 (부호 무시) 
and     / &     /비트 AND
or      / |     /비트 OR
xor     / ^     / 비트 XOR
inv()   / ~     / 비트 반전
